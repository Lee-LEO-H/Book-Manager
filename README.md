# 图书管理系统
#### 该项目仅使用了c++语言，代码逻辑简单清晰，对初学者十分友好。这是本人面向对象程序设计课程的期末大作业，基本满足了所有图书管理所需的功能。因其不需要过于复杂的输出界面，故该系统采用的是控制台交互的方式呈现，其次该项目支持记事本存储读取数据。

#### 实现功能有：
  * 添加人员和图书信息
  * 删除人员和图书信息
  * 修改人员和图书信息
  * 查询人员和图书信息及借阅归还记录
  * 借阅书籍
  * 归还书籍
